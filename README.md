# Python Version
Use Python version 3.11 is the best
# Install Library
\```
cd app
pip install -r requirements.txt
\```
# Run code
Make sure you are in app/ folder
\```
gradio gradio_demo.py
\```
And then go to : http://127.0.0.1:7860
